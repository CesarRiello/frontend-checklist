frontend-checklist
==================

Um simples checklist de frontend

* Dar suportar
  * Browser
  	* Chrome
  	* Internet Explorer
  	* Firefox
  	* Safari
  * OS
  	* Windows
  	* Mac OSX
  	* iOs
  	* Android
  	* Linux
  * Device
  	* Desktop - Laptop
  	* Tablet
  	* Mobile
* Validar
  * HTML
  * CSS
  * Javascript
* Checagem de erros
  * Images
  * CSS
  * Javascript
  * links
* SEO
  * Estrutura HTML
  * Keywords na página
  * Meta description: até 155 caracteres
  * Title: até 65-59 caracteres
* Otimização
  * Images
  	* Sprites
  * CSS
  	* Concatenação
  	* Minificação
  	* No header da página
  * Javascript
  	* Concatenação
  	* Minificação
  	* No fim da página
